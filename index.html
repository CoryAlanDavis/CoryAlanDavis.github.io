Latest Projects

Compound

Compound is a game created using Unreal Engine 4. I worked on gameplay elements such as player abilities, Artificial intelligence, environmental hazards and traps. Worked in a team of ~30 programmers and artists.

Mock Website

This was a project to create a simple social media website that had all the basic fuctionality. Users can create an account, make posts, add friends, and upload their own profile picture! The website made use of cookies, CherryPy, Mako and SQLite. Worked in a team of two.

A* Algorithm Maze

This small project used an A* algorithm to find a path between two different points in a maze. The user of the program has the ability to choose the two points. A Navigation Mesh was used for the maze and lines were used to show the path the algorithm took.

Cellular Automaton Based Heat Simulation

This small project aimed to create a simplified simulation of heat spreading through the air. This project used a cellular automaton based heat equation. I used Python and Pygame to create this project.
